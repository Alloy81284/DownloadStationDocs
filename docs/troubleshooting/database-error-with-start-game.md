# 开始游戏时数据库错误

在运行P社游戏时候，有时候会出现DLC和DLC补丁都打齐全的情况下DLC仍然不生效的情况，并且会提示你的数据库报错。

![数据库错误截图](https://img.alloy.eu.org/1751550307-6314970d40634023-073fa66b-ba9d-4f26-89dd-7952b3ae4489.png)

## 解决办法

报错通常由启动器文档文件引起：

- **清除启动器数据**：在启动器选项里清除所有数据缓存。
- **清空对应游戏文档**：直接到你的文档文件夹中删除整个游戏文档文件夹。也就是到这个报错上面说的地方，直接把HOI4/EU4/CK3/VIC3等对应报错游戏文件夹直接全删除，再重新启动游戏。

---

## 详细步骤

### 方法一：清除启动器缓存

1. 打开P社启动器
2. 进入设置选项
3. 找到"清除缓存"或"重置数据"选项
4. 重启启动器

### 方法二：删除游戏文档文件夹

1. 打开文件资源管理器
2. 导航到 `文档\Paradox Interactive\`
3. 找到对应的游戏文件夹（如HOI4、EU4等）
4. **备份存档文件**（通常在save_games文件夹中）
5. 删除整个游戏文件夹
6. 重新启动游戏

## 注意事项

> **⚠️ 重要提醒**
>
> 删除游戏文档文件夹会清除：
> - 游戏设置
> - MOD设置
> - 存档文件（请务必备份！）
> - 自定义配置
