# 常见问题解答

这里汇总了社区内最常见的问题并提供解决方案，无需等待即可自行排查。

## 🔧 问题分类

### 游戏启动问题

#### [开始游戏时崩溃报错](error-with-start-game.md)
游戏启动时出现错误对话框，通常是缺少运行库导致。

**关键词**: 崩溃、DLL错误、运行库缺失

**解决方案预览**:
- 安装Visual C++ Redistributable
- 使用DirectX Repair工具修复

#### [开始游戏时数据库错误](database-error-with-start-game.md)
DLC已安装但不生效，启动器显示数据库相关错误。

**关键词**: 数据库错误、DLC不生效、启动器问题

**解决方案预览**:
- 清除启动器缓存
- 删除游戏文档文件夹

### MOD相关问题

#### [学习版安装MOD](install-mod.md)
详细的MOD安装步骤和常见安装问题解决方案。

## 💡 故障排除流程

### 步骤1：基础检查
1. 确认游戏文件完整性
2. 检查系统运行库是否完整
3. 验证游戏版本与MOD兼容性

### 步骤2：日志分析
1. 查看游戏错误日志
2. 检查启动器日志文件
3. 记录具体错误信息

### 步骤3：逐步排除
1. 禁用所有MOD测试
2. 重置游戏配置
3. 重新安装问题组件

### 步骤4：寻求帮助
如果以上步骤都无法解决问题：
- 加入QQ群
- 提供详细的错误信息和系统配置
