# 开始游戏时崩溃报错

在运行P社游戏时候，可能出现崩溃报错，那应该是你少安装了依赖。

![游戏崩溃错误截图](https://img.alloy.eu.org/1751550242-da5b099314c4db21-a2fbf71d-d23b-48a8-b3e2-c680cd6c33a4.png)

## 解决办法

报错通常由缺失运行库或DLL文件引起：

- **运行库**：在下载站里下载游戏所需运行库（如Visual C++、.NET Framework等）。
- **DLL修复**：在下载站里下载使用工具 DirectX Repair，点击"检测并修复"即可。
