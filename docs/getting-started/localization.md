# 关于汉化

目前，P社游戏本体除去欧陆风云4和其他前作老游戏，主流P社游戏已经完全汉化。

## 本体切换汉化

- 启动器选择中文即可

![启动器语言设置](https://img.alloy.eu.org/1751560231-ad148b84b4dd5e03-91b84f43-bda1-426b-ae53-fdddc9e0e0be.png)

## 欧陆风云4切换汉化

- 必须打上双字节补丁后再启用汉化MOD才能汉化成功

## 部分MOD汉化问题

由于早期部分游戏本身没有中文但是后续出了中文，导致部分MOD汉化原本还在英文本地化文件没有切换到中文本地化文件。部分MOD要求启动器可能要选择英文启动，请务必按照MOD要求来执行，否则可能导致本地化失效。
