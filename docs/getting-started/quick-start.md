# 快速开始：通用安装指南

本指南将为您提供一个通用的资源安装流程。针对特定的游戏或补丁，请务必参考其独立的详细说明页面。

## 准备工作

在开始之前，请确保您已经完成了以下准备工作。

> **必要软件**
>
> - 请确保您的系统已安装最新的 **Visual C++ Redistributable** 和 **.NET Framework**。
> - 我们推荐使用 [7-Zip](https://www.7-zip.org/) 或 [Bandizip](https://www.bandisoft.com/bandizip/) 进行文件解压，以避免不必要的错误。

---

## 安装步骤

以下是标准的安装流程，适用于大部分 MOD 和补丁。

### 1. 下载与解压

从本站下载您需要的压缩包文件。下载后，使用解压软件将其解压到一个**不包含中文路径**的临时文件夹中。

### 2. 备份源文件

这是最重要的一步！

> **⚠️ 务必备份！**
>
> 在覆盖任何游戏文件之前，**请务必备份您的游戏存档和即将被覆盖的原始文件！** 这将确保在出现问题时，您可以轻松地恢复到原始状态。

### 3. 覆盖文件

仔细观察对应根目录文件结构，将对应文件或文件夹复制到游戏根目录。

### 4. 验证

启动游戏，检查 MOD 或补丁是否已生效。如果遇到问题，请首先查阅"常见问题解答"部分。
