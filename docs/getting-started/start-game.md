# 启动游戏

## 安装启动器

- 第一次在此电脑打开P社游戏时，需要安装启动器。
- 解压游戏本体及DLC文件后，我们可以得到类似这样的文件夹，我们需要先点击launcher-installer安装P社启动器。

![启动器安装](https://img.alloy.eu.org/1751558919-9c3348ce5a6daf67-9a13fbd4-2eda-4170-bf21-e41145f5b71b.png)

- 之后我们点击dowser.exe启动游戏。

## 如果你已经安装过启动器就可以忽略上一步

- 直接点击dowser.exe启动游戏。
